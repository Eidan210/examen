# 📚 ACME School - Gestor de Exámenes

## ¿Qué es este proyecto?

ACME School es una aplicación web para gestionar exámenes educativos. Permite a los administradores crear y organizar pruebas, mientras que los estudiantes pueden resolver exámenes de forma interactiva.

## Características principales

- **🔐 Autenticación**: Sistema de login seguro para usuarios autorizados
- **📝 Crear Exámenes**: Portal para diseñar y configurar nuevas pruebas
- **✍️ Resolver Exámenes**: Módulo interactivo para que los estudiantes resuelvan exámenes con contador de tiempo

## Estructura del proyecto

```
├── login oc/              → Módulo de autenticación
├── PortalCreaciónExamen/  → Herramienta para crear exámenes
└── ResolverExamen-contador/ → Módulo para resolver exámenes
```

## 🚀 Cómo empezar

Para probar la aplicación, abre el archivo:

```
Proyecto JavaScript/login oc/index.html
```

Desde allí podrás acceder a todas las funcionalidades del sistema.
